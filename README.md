A Photo CD (.pcd) “Player” created specifically for use with the redump.org’s multi-bin / cue standard.

Displays images and plays redbook audio, does not support CD-i menu system. Export images to .png.

Rom sets available on archive.org.

Power words: Kodak Photo CD, Portfolio, emulator, pcd.
